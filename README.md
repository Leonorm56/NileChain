## Overview

**PearlFarmer** is a premium automation tool for Telegram Mini-Apps (built for crypto “farming” workflows). It supports flexible deployment as a Chrome Extension or Progressive Web App (PWA), with optional multi-account and cloud backends.

### Key Features

- Multi-bot automation for Telegram Mini-Apps
- Chrome Extension and PWA builds
- Local/Cloud session management
- Proxy support
- Real-time monitoring

---

## Quick Start

### Prerequisites

- Node.js v18+
- pnpm
- Chrome/Chromium

### Installation

```bash
git clone https://github.com/TODO_ORG/TODO_REPO.git
cd TODO_REPO
pnpm install
```

---

## Build (Chrome Extension)

```bash
pnpm build:farmer
```

Outputs are produced under `apps/purrfect-farmer/` (see `apps/purrfect-farmer/README.md` for details).
